### Mecânicas
 O jogo consiste onde dois personagens que serão constrolados pelo jogador um pelo botões direcionais e outros pelos e outro pelo 
 botões wsad no teclado. Andando pelo cenário a procura de outro jogador para jogar uma bomba e esperar que os jogadores sejam atingidos
 pelo fogo da explosão e quem conseguir atingir o jogador com a bomba será declarado o vencendor. Infelizmente não usada nenhuma técnica de algoritmo
 de I.A neles.
